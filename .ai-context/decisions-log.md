# Decisions Log

> Decisiones técnicas que surjan durante la implementación y que **aún no estén en los docs principales** (`04-arquitectura-tecnica.md`).
> Una vez una decisión está aquí varias veces o afecta a varios módulos, **se promueve a un ADR** en el doc 04.

## Formato

```
## YYYY-MM-DD · Título de la decisión

**Contexto:** qué situación llevó a la decisión
**Opciones consideradas:** A, B, C
**Decisión:** qué se eligió
**Razón:** por qué
**Próxima revisión:** cuándo o si nunca
```
